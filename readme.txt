This code is very simple to run as it reads the input
from the user directly instead of using an input file. It can
be compiled through an online python compiler or Visual studio code.

Example input
Enter the start state matrix:
1 3 4
8 0 6
7 5 2
Enter the goal state matrix:
1 2 3
8 0 4
7 6 5
BFS found the shortest path
shortest path cost: 8
Number of moves: 8
Shortest path:
1 2 3
